# ESPro+

A Chrome extension that enhances the Christ University ESPro attendance dashboard.

## Features

- 🎯 75% Attendance Goal Calculator
- 🎯 85% Attendance Goal Calculator
- 💤 Safe Bunks Calculator
- 📈 Attendance Simulator
- ⚡ Lightweight and fast
- 🔒 Runs completely in your browser

## Installation

1. Download this repository.
2. Extract the ZIP (if downloaded as a ZIP).
3. Open `chrome://extensions`.
4. Enable **Developer mode**.
5. Click **Load unpacked**.
6. Select the `ESPro-Plus` folder.
